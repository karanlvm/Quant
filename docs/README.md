# GitHub Pages Files

This directory contains the static files for GitHub Pages deployment.

Files are automatically generated by the GitHub Actions workflow:
- `index.html` - Bloomberg Terminal interface
- `portfolio.json` - Portfolio allocations
- `market-data.json` - Market data
- `sentiment.json` - News sentiment scores
- `performance.json` - Performance metrics
- `stats.json` - System statistics

Do not edit these files manually - they are generated automatically!
